Diffusion Models

This project explores DeepFloyd IF diffusion models for text-to-image generation, image editing, and creative applications. We implement sampling loops, denoising processes, and advanced techniques like classifier-free guidance, inpainting, visual anagrams, and hybrid images.

The diffusion model successfully captures the creative and surreal elements described in each prompt. 1.2.3 Training Analysis

Detailed analysis of the training behavior and model limitations.

Epoch 1
Epoch 5
Training Curve

Observations

Pattern: Regardless of the input noise or number of training epochs, all generated outputs are highly similar. The denoised images appear as a blurry superposition of all digits (0-9), with overlapping contours from different digit classes visible.

Explanation

Since the input is pure random noise (containing no information about which digit to generate), but each batch contains different target images (digits 0-9), the model learns to output a single image that minimizes the average MSE across all training samples.

Key insight: With MSE loss, the optimal prediction is the centroid (mean) of the training dataset ‚Äî a blurry "average digit" that represents the center of the data distribution.

Why this happens:

  • The model cannot distinguish which digit to generate from random noise
  • To minimize loss across all training images, it outputs their average
  • This is mathematically equivalent to a clustering centroid

Conclusion

A single-step denoiser cannot perform generation tasks. It learns the dataset mean rather than the underlying distribution, failing to produce diverse, recognizable digits. This motivates the need for more sophisticated approaches like Flow Matching with time conditioning.
